# 01 — Instalação

```bash
composer require argws/asaas-sdk-php
```

## Requisitos

- PHP 8.1+
- ext-json

## Atualização de versões

O Packagist publica versões via **tags Git**. Para atualizar:

```bash
composer update argws/asaas-sdk-php
```

Você pode fixar versões com SemVer:

```json
{
  "require": {
    "argws/asaas-sdk-php": "^0.1"
  }
}
```
